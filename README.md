###Full Stack Authentication System

A full-stack authentication system built using React, Node.js, Express, and MongoDB that allows users to securely register, log in, and access protected routes using JWT-based authentication.
The system implements secure password hashing using bcrypt and ensures only authenticated users can access protected resources like the user dashboard.

##Features :

User Registration
User Login
Secure Password Hashing using bcrypt
JWT-based authentication
Protected API routes
User Dashboard
Logout functionality
React-based frontend interface
